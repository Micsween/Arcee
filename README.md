# Arcee
This is an AI chatbot that will help you plan and track your workout routine. 

## Arcee is an AI personal trainer that can do the following:
- Track your progress, including: reps, weights, sets
- Help you plan a workout routine (AI chatbot)
- Predict where you will be after a certain time period


For the database, I will use MongoDB.
To host, I will use Caddy and Amazon AWS. This will also let me pick a domain name.
To code the website, I will use React or Svelte.
To code the AI, I will use Python or C#. I will also be using Claude to assist me.
